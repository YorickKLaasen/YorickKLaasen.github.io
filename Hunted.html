About the Project

What is Hunted?

Hunted is a 2D top-down island survival game made in Unity 6.3. You spawn on an island and need to open chests to get weapons to kill enemies that spawn in waves each wave becoming more difficult.

Genre

Version History

Patch Notes

Where you can see things that have been added or fixed.

v0.3.0 Latest June 18th 2026

Added

  • Weapon Modifiers - There are different modifiers weapon can get that either increase stats or change bullets
  • Enemy AI - Flowfield AI for enemies.
  • Armored Enemy - An Enemy that has a hitshield that needs to be broken before taking damage.
  • Wave System - Waves have a budget to spawn enemies that increases every wave.
  • DamageText - Damage Text that shows how much damage you dealt to an enemy.
  • WeaponChest Dropchance - Items with higher rarity get an increased dropchance from higher graded chests.
  • Player Healthbar - Players can now see how much health they have left.
  • Player XPbar - Players can now get XP from killing enemies that levels them up
  • Main Menu - On starting the game the player sees the Main Menu before the game.
  • Game Over - On player death opens the GameOver panel and shows Player stats

Changes

  • Updating Weapon Stats - now done via a seperate script instead of Prefab
  • Weapon Inventory UI - added weapon names to inventory UI so players know which modifier and weapon they have.

Fixed

  • No Weapon From Chest - players sometimes didnt get weapons from a chest
  • Enemy Attack Cooldown - Enemies could'nt attack the player a 2nd time until player left enemy collider.
v0.2.0 Stable May-June 2026

Added

  • Weapon Inventory - Weapon inventory has been added where player can equip their weapons.
  • Weapon Shoot - Weapons can now shoot with some weapons having piercing bullets.
  • Test Enemy - Added test enemy to see if bullets damage enemies.
  • Chest Loot - Chests can now give the player weapons.
  • Ammo UI - Added UI that shows how much ammo the gun currently has.
  • Player Aim/ Gun Rotate - weapons now rotate with player Mouse position.
v0.1.0 May 2026

Added

  • Player Movement - The player can now move around the Map with the W,A,S,D keys.
  • Island Map - The island for the map has been generated.
  • Chest Opening - chests can be opened but don't do anything yet.
